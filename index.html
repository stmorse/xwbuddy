Crossword Buddy is a tool designed to help you solve crosswords without directly giving away the answers. It provides alternative clues to help you think differently about the puzzle.

Alternative clues are AI-generated (currently using Llama 3.1 (70B)) using custom prompts and some coaxing.

This project is open-source -- come be a contributor!
